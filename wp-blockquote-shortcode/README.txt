# Wp Blockquote Shortcode

Webページ内の文を選択しショートコードを実行するだけで、手軽に引用文のショートコードを作成します。

## インストール

ZIPをダウンロードして解凍、中身を開きます。

### プラグイン

/wp-blockquote-shortcode/wp-blockquote-shortcode

フォルダを、WordPressプラグインフォルダに入れて、プラグインを有効にします。

### ブックマークレット

/wp-blockquote-shortcode/bookmarklet/wp-blockquote-shortcode-bookmarklet.html

をブラウザで開き、リンクをブックマークバーにドラッグします。


## 使い方

任意のWebページで、引用したい文を選択し、そのままブックマークレットを実行します。

ショートコードが生成されたダイアログが表示されるので、内容をWordPress本文にコピペします。
